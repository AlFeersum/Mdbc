
TOPIC
    about_Mdbc

SHORT DESCRIPTION
    MongoDB cmdlets based on the official C# driver.

LONG DESCRIPTION
    The module provides the following cmdlets and tools:

    Connect, set the default server, database, and collection:

        Connect-Mdbc

    Add, get, remove, and update documents:

        Add-MdbcData
        Get-MdbcData
        Remove-MdbcData
        Update-MdbcData

    Create new documents for Add-MdbcData, Export-MdbcData:

        New-MdbcData

    Create update expressions for Update-MdbcData:

        New-MdbcUpdate

    Create queries for Get-MdbcData, Update-MdbcData, and etc.:

        New-MdbcQuery

    Add collections with special properties, e.g. capped:

        Add-MdbcCollection

    Invoke map and reduce procedure:

        Invoke-MdbcMapReduce

    Invoke MongoDB command including not covered by Mdbc or C# driver:

        Invoke-MdbcCommand

    File based data persistence even without MongoDB installed:

        Export-MdbcData
        Import-MdbcData

    Interactive profile adds more features for interactive sessions:

        Mdbc.ps1
